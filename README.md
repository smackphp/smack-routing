## Install

Via Composer

``` bash
$ composer require smackphp\smack-routing
```

## Usage

``` php

require('vendor/autoload.php');

$router = new \Smack\Routing\Router;
$router->add('GET', '/^\/user\/(?<id>\d+)$/', function($id){
	echo 'hello #'.$id;
});

$match = $router->route(
	$_SERVER['REQUEST_METHOD'], 
	parse_url(
	    $_SERVER['REQUEST_URI'], 
	    PHP_URL_PATH
	)
);

call_user_func_array(
	$match->getHandler(), 
	$match->getParams()
);
```
